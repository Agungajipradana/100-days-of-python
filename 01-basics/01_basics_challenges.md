# Python Basics Challenges

## Easy Level

1. **Print your name**
   Create a variable called `name` with your name, then print it using `print()`.

2. **Print multiple values**
   Print your name, age, and city in one `print()` statement, separated by commas.

3. **Add a comment**
   Write a single-line comment above a line of code explaining what that line does.

4. **Check the type**
   Create a variable with a number and another with text, then use `type()` to print both of their data types.

5. **Say hello using f-string**
   Create a variable called `name`, then print `"Hello, <name>!"` using an f-string.

---

## Medium Level

1. **Multiple assignment**
   Create three variables `x`, `y`, `z` in a single line and assign them the values `10`, `20`, `30`. Print all three in one line.

2. **Dynamic typing demo**
   Create a variable, assign it a number, print its type, then reassign it with a string and print its type again.

3. **Custom separator**
   Use `print()` with the `sep` parameter to print `"Python-is-awesome"` from three separate words: `"Python"`, `"is"`, `"awesome"`.

4. **Same line output**
   Use the `end` parameter in `print()` so that two separate `print()` calls appear on the same line.

5. **Constant convention**
   Create a variable named `MAX_SCORE` following the ALL_CAPS convention, and explain in a comment why Python doesn't have true constants.

---

## Hard Level

1. **Variable swap without a third variable**
   Create two variables `a = 5` and `b = 10`, then swap their values without using a temporary/third variable. Print the result before and after.

2. **Build a simple profile card**
   Create variables for `name`, `age`, `city`, and `hobby`. Using only f-strings and `print()`, display a small "profile card" output like:

    ----- Profile -----
    Name: ...
    Age: ...
    City: ...
    Hobby: ...

3. **Type-safe input check**
   Create a variable `value = "25"` (a string). Convert it into an integer, add `5` to it, then print the result along with the type of the final value.
